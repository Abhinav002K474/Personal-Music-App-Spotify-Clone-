# 🎵 PARADISE 音楽: Where Music Meets Anime

PARADISE 音楽 is a premium, high-fidelity music streaming platform designed with a futuristic "Electric Blue & Black" aesthetic. It combines immersive AMV/GMV visualizers with advanced audio processing tools to create a professional-grade listening sanctuary.

![PARADISE 音楽 UI Preview](https://res.cloudinary.com/dhocv2p3t/image/upload/v1778179316/download_b4rtyo.jpg)

## ✨ Key Features

### 🎧 Immersive Audio Engine
- **3-Band Live Equalizer**: Tweak Bass, Mids, and Highs in real-time using a professional BiquadFilter engine.
- **Surround Sound Toggle**: Instantly activate a custom V-curve soundstage for a wider, more cinematic experience.
- **Bi-Directional Sync**: EQ controls are synchronized between the sidebar and the full-screen "Now Playing" overlay.

### 🌌 Visual Excellence
- **High-Fidelity Visualizers**: Integrated AMV (Anime Music Video) and GMV (Game Music Video) backgrounds synchronized with your tracks.
- **Dual-Video Sync**: A unique dual-video system that keeps the center card and background atmosphere perfectly in time.
- **Glassmorphism UI**: A stunning, ultra-blur interface with glowing Electric Blue accents and smooth micro-animations.

### 📱 Fully Responsive
- **Desktop Sidebar**: A powerful management center with live EQ and quick navigation.
- **Mobile Bottom Nav**: A thumb-friendly navigation bar for on-the-go listening.
- **Adaptive Player**: An intelligent player bar that reconfigures itself for any screen size.

### 📡 Advanced Functionality
- **Cast Portal**: A realistic "Cast to Device" interface with a ripple-scanning effect for WiFi device discovery.
- **Dynamic Library**: Real-time search, sorting, and track management with persistent storage.
- **Secure Access**: Protected by a high-fidelity authentication layer.

## 🚀 Technologies Used
- **Core**: Vanilla HTML5, CSS3, and JavaScript (ES6+).
- **Audio**: Web Audio API (AudioContext, BiquadFilterNodes, AnalyserNode).
- **Visuals**: HTML5 Video Sync, Canvas API for mini-visualizers.
- **Storage**: Browser LocalStorage for library persistence.
- **Branding**: Powered By Immortal Anime.
